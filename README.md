AI memos freshly typeset from source.

Many MIT AI memos were written using Richard Greenblatt's TJ6
formatting and typesetting program.  Out of those, many were printed
on Xerox experimental "Xerographic Printer", the XGP.  The original
source files were retrieved from MIT's Tapes of Tech Square backup
collection.  The TJ6 program was run on a PDP-10 emulator, and the
output was sent to a PDP-11 emulator running the original XGP driver
code.
